** Connect to MySQL instance

mysql -u root -p'root' -h $IP -P 3306 --skip-ssl-verify-server-cert

** Enumerate
| Description                 | Query                                                                      |
|-----------------------------+----------------------------------------------------------------------------|
| Version                     | ~SELECT version();~                                                        |
| Current User                | ~SELECT system_version();~                                                 |
| All Users                   | ~SELECT * FROM mysql.user;~                                                |
| Databases                   | ~SHOW databases;~                                                          |
| Tables of Database ~offsec~ | ~SELECT * FROM information_schema.tables WHERE table_schema = 'offsec'~    |
| Passwords of specific user  | ~SELECT user, authentication_string FROM mysql.user WHERE user = 'offsec'~ |

Use *\G* instead of *;* to format output for console e.g., ~SELECT version()\G~.
