* Navigation

** Cursor Movement

| Key       | Action                      |
|-----------+-----------------------------|
| C-f / C-b | Forward / backward one char |
| M-f / M-b | Forward / backward one word |
| C-a / C-e | Line start / end            |

** Jump to position

| Key   | Action                       |
|-------+------------------------------|
| M-<   | Jump to beginning of buffer  |
| M->   | Jump to end of buffer        |
| M-g g | Jump to specific line number |

* Editing

** Cut / Copy / Paste

| Key | Action                              |
|-----+-------------------------------------|
| C-w | Cut (kill) region                   |
| M-w | Copy (kill-ring-save)               |
| C-y | Paste (yank)                        |
| M-y | Cycle through earlier kills         |
| M-d | Delete word forward (like Vim ~cw~) |

** Setting the mark (selecting text)

| Key           | Action                       |
|---------------+------------------------------|
| C-Space / C-@ | Set start of mark            |
| C-g           | Cancel / deactivate the mark |

** Kill (delete) line

| Key                 | Action                            |
|---------------------+-----------------------------------|
| C-a C-k C-k         | Delete whole line (like Vim ~dd~) |
| C-k                 | Kill from cursor to end of line   |
| M-x kill-whole-line | Delete entire line at cursor      |

After killing a line move to the desired location and use ~C-y~ to paste it.

** Undo / redo

| Key | Action              |
|-----+---------------------|
| C-/ | Undo last operation |
| C-? | Redo last operation |

* Window Management
# Emacs terminology: Frame = OS window, Window = split pane, Buffer = file/content

** Frames (OS windows)

| Key     | Action                |
|---------+-----------------------|
| C-x 5 2 | Create a new frame    |
| C-x 5 0 | Close current frame   |
| C-x 5 o | Switch to next frame  |

** Windows (split panes)

| Key   | Action                                 |
|-------+----------------------------------------|
| C-x 2 | Split window horizontally (top/bottom) |
| C-x 3 | Split window vertically (left/right)   |
| C-x 0 | Close current window                   |
| C-x 1 | Close all other windows (maximise)     |
| C-x o | Switch to next window                  |

** Buffers

| Key     | Action                               |
|---------+--------------------------------------|
| C-x C-f | Open / create a new file in a buffer |
| C-x b   | Switch to a buffer (prompts by name) |
| C-x C-b | List all open buffers                |
| C-x k   | Kill (close) current buffer          |

* Dired (File Manager)

** Opening & Closing

| Key     | Action                             |
|---------+------------------------------------|
| C-x d   | Open Dired (prompts for directory) |
| C-x C-f | Open Dired (prompts for file)      |
| C-x C-j | Jump to Dired in current directory |
| q       | Close Dired buffer                 |

** File Operations

| Key       | Action                                      |
|-----------+---------------------------------------------|
| +         | Create a new directory                      |
| C-x C-f   | Create a new file (open for editing)        |
| m         | Mark a file                                 |
| t         | Toggle mark (invert selection)              |
| u         | Unmark a file                               |
| U         | Unmark all files                            |
| d         | Mark file(s) for deletion                   |
| C         | Copy marked file(s)                         |
| D         | Delete file immediately (no mark required)  |
| R         | Rename / move a file                        |
| x         | Execute deletion (confirm marked file(s))   |
| % m REGEX | Mark files matching a regex pattern         |
