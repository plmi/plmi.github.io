* Navigation

** Cursor Movement

| Key       | Action                      |
|-----------+-----------------------------|
| C-f / C-b | Forward / backward one char |
| M-f / M-b | Forward / backward one word |
| C-a / C-e | Line start / end            |

** Jump to position

| Key   | Action                       |
|-------+------------------------------|
| M-<   | Jump to beginning of buffer  |
| M->   | Jump to end of buffer        |
| M-g g | Jump to specific line number |

* Editing

** Cut / Copy / Paste

| Key     | Action                              |
|---------+-------------------------------------|
| C-w     | Cut (kill) region                   |
| M-w     | Copy (kill-ring-save)               |
| C-y     | Paste (yank)                        |
| M-y     | Cycle through earlier kills         |
| M-d     | Delete word forward (like Vim ~cw~) |
| S-Mouse | Select text in termal mode          |

** Setting the mark (selecting text)

| Key           | Action                       |
|---------------+------------------------------|
| C-Space / C-@ | Set start of mark            |
| C-g           | Cancel / deactivate the mark |

** Kill (delete) line

| Key                 | Action                            |
|---------------------+-----------------------------------|
| C-a C-k C-k         | Delete whole line (like Vim ~dd~) |
| C-k                 | Kill from cursor to end of line   |
| M-x kill-whole-line | Delete entire line at cursor      |

After killing a line move to the desired location and use ~C-y~ to paste it.

** Undo / redo

| Key | Action              |
|-----+---------------------|
| C-/ | Undo last operation |
| C-? | Redo last operation |

* Window Management
# Emacs terminology: Frame = OS window, Window = split pane, Buffer = file/content

** Frames (OS windows)

| Key     | Action                |
|---------+-----------------------|
| C-x 5 2 | Create a new frame    |
| C-x 5 0 | Close current frame   |
| C-x 5 o | Switch to next frame  |

** Windows (split panes)

| Key   | Action                                 |
|-------+----------------------------------------|
| C-x 2 | Split window horizontally (top/bottom) |
| C-x 3 | Split window vertically (left/right)   |
| C-x 0 | Close current window                   |
| C-x 1 | Close all other windows (maximise)     |
| C-x o | Switch to next window                  |

** Buffers

| Key     | Action                               |
|---------+--------------------------------------|
| C-x C-f | Open / create a new file in a buffer |
| C-x b   | Switch to a buffer (prompts by name) |
| C-x C-b | List all open buffers                |
| C-x k   | Kill (close) current buffer          |

** Bookmarks

| Key     | Action                        |
|---------+-------------------------------|
| C-x r m | Set bookmark for current file |
| C-x r b | Jump to a bookmark            |
| C-x r l | List all bookmarks            |
| C-x r d | Delete a bookmark             |

* Org Mode

** Structure templates (type then ~Tab~ to expand)

| Shortcut | Expands to                            |
|----------+---------------------------------------|
| <s       | #+BEGIN_SRC ... #+END_SRC             |
| C-c '    | Edit source block in dedicated buffer |
| <e       | #+BEGIN_EXAMPLE ... #+END_EXAMPLE     |
| <q       | #+BEGIN_QUOTE ... #+END_QUOTE         |
| <c       | #+BEGIN_CENTER ... #+END_CENTER       |
| <l       | #+BEGIN_EXPORT latex ... #+END_EXPORT |
| <h       | #+BEGIN_EXPORT html ... #+END_EXPORT  |
| <a       | #+BEGIN_EXPORT ascii ... #+END_EXPORT |
| <c       | #+BEGIN_COMMENT ... #+END_COMMENT     |

Note: requires ~(require 'org-tempo)` in config in Emacs 26+.

** Headings

| Key           | Action                                    |
|---------------+-------------------------------------------|
| M-RET         | Insert new heading at same level          |
| M-RIGHT       | Demote heading (e.g. * to **)             |
| M-LEFT        | Promote heading (e.g. ** to *)            |
| M-UP / M-DOWN | Move heading up / down                    |
| Tab           | Cycle fold state of current heading       |
| S-Tab         | Cycle fold state of all headings globally |
| C-c C-n       | Jump to next heading                      |
| C-c C-p       | Jump to previous heading                  |

** Tables

| Key              | Action                           |
|------------------+----------------------------------|
| Tab              | Move to next cell                |
| S-Tab            | Move to previous cell            |
| RET              | Move to next row                 |
| M-RET            | Insert new row below             |
| C-c C-c          | Re-align table without moving    |
| C-c -            | Insert horizontal separator row  |
| C-c ^            | Sort table by current column     |
| M-UP / M-DOWN    | Move current row up / down       |
| M-LEFT / M-RIGHT | Move current column left / right |

** Links

| Key     | Action                         |
|---------+--------------------------------|
| C-c C-l | Insert or edit link            |
| C-c C-o | Open link at cursor            |

** Images / Commands

| Shortcut        | Action                                          |
|-----------------+-------------------------------------------------|
| ~[[./foo.png]]~ | Insert image                                    |
| C-c i s         | Insert latest image from ./screenshots (custom) |
| C-c i i         | Insert image from clipboard (custom)            |
| C-c i c         | Insert formatted command output (custom)        |

Insert images with caption
#+begin_src text
#+CAPTION: Your caption
#+NAME: fig:yourimage
#+ATTR_HTML: :width 900px
[[./foo.png]]
#+end_src

** Todo & Agenda

| Key         | Action                                  |
|-------------+-----------------------------------------|
| C-c C-t     | Cycle TODO state (TODO -> DONE -> none) |
| C-c C-d     | Set a deadline                          |
| C-c C-s     | Set a schedule date                     |
| C-c C-x C-i | Clock in (start timing a task)          |
| C-c C-x C-o | Clock out                               |
| C-c a       | Open the agenda dispatcher              |

** Export

| Key         | Action                    |
|-------------+---------------------------|
| C-c C-e     | Open export dispatcher    |
| C-c C-e h h | Export to HTML file       |
| C-c C-e l p | Export to PDF (via LaTeX) |
| C-c C-e t u | Export to plain text      |

* Dired (File Manager)

** Opening & Closing

| Key     | Action                             |
|---------+------------------------------------|
| C-x d   | Open Dired (prompts for directory) |
| C-x C-f | Open Dired (prompts for file)      |
| C-x C-j | Jump to Dired in current directory |
| q       | Close Dired buffer                 |

** File Operations

| Key       | Action                                              |
|-----------+-----------------------------------------------------|
| +         | Create a new directory                              |
| C-x C-f   | Create a new file (open for editing)                |
| m         | Mark a file                                         |
| t         | Toggle mark (invert selection)                      |
| u         | Unmark a file                                       |
| U         | Unmark all files                                    |
| d         | Mark file(s) for deletion                           |
| C         | Copy marked file(s)                                 |
| D         | Delete file immediately (no mark required)          |
| R         | Rename / move a file                                |
| x         | Execute deletion (confirm marked file(s))           |
| % m REGEX | Mark files matching a regex pattern                 |
| W         | Runs ~browse-url-of-file~ and opens file in browser |
